# 👨🏻‍💻 Customer Behavior Analysis Project

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. It focuses on analyzing customer shopping behavior, identifying trends, and generating actionable business insights.

## 📌 Project Overview

The main objective of this project is to transform raw customer data into meaningful insights through different stages of data analysis:

✅ **Data Cleaning & Preparation (Python)**  
- Import datasets  
- Handle missing values  
- Clean and transform data  
- Perform exploratory data analysis  

✅ **Business Insights (SQL)**  
- Analyze customer purchasing patterns  
- Identify top-performing segments  
- Evaluate loyalty and retention trends  
- Solve business questions using SQL queries  

✅ **Dashboard & Visualization (Power BI)**  
- Build an interactive dashboard  
- Visualize customer behavior trends  
- Present KPIs and performance metrics  

---

## 🛠️ Tools Used

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Jupyter Notebook

## 📂 Project Files
